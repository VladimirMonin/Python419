Знакомство с CSS

Определение CSS

CSS (Cascading Style Sheets) - это язык описания внешнего вида документа, написанного на HTML. Он позволяет отделить содержание документа от его оформления, что делает код более чистым и удобным для поддержки. При условии что стили лежат в отдельном файле.

Подключение CSS. 3 способа. 🧐

Существует 3 способа подключения CSS к HTML-документу:

  • Инлайн стиль (inline style) - стили прописываются непосредственно в HTML-тегах с помощью атрибута style. Например: <h1 style="color: red;">Заголовок</h1>
  • Внутренний стиль (internal style) - стили прописываются документа в теге head с помощью тега style.
  • Внешний стиль (external style) - стили прописываются в отдельном файле с расширением .css, который подключается к HTML-документу с помощью тега link в разделе head. Например: <link rel="stylesheet" href="styles.css">

Типы селекторов

Сегодня мы будем смотреть на селекторы по тегу вроде h1 и по классу вроде .class

  • h1 - селектор по тегу. Воздействует на все указанные теги.
  • h2, code - селектор по тегам, воздействует на указанные теги.
  • li>code - селектор по тегу и вложенному тегу. Воздействует на указанные теги, которые находятся внутри указанных тегов.
  • .special - селектор по классу. Воздействует на все элементы с указанным классом.
  • code.special - селектор по тегу и классу. Воздействует на указанные теги с указанным классом.
