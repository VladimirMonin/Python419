Что такое блочная модель?

Блочная модель CSS - это фундаментальная концепция, определяющая как элементы отображаются на веб-странице. Каждый HTML-элемент представляет собой прямоугольный блок, который состоит из четырех основных частей:

  • Содержимое (content) - область, где отображается текст, изображения и другой контент
  • Внутренние отступы (padding) - прозрачная область вокруг содержимого
  • Границы (border) - рамка вокруг padding и содержимого
  • Внешние отступы (margin) - прозрачная область за пределами границы

Принципы расчета отступов

Отступы (margin и padding) рассчитываются по следующим правилам:

  • Процентные значения - рассчитываются от ширины родительского элемента
  • Абсолютные значения (px, em, rem) - используются как есть
  • Auto значение - автоматически рассчитывается браузером

Пример расчета процентных отступов:

.parent {
    width: 600px;
}

.child {
    margin: 10%; /* 60px (10% от 600px) */
    padding: 5%; /* 30px (5% от 600px) */
}

Общая ширина элемента рассчитывается по формуле:

ширина = margin-left + border-left + padding-left + 
        width + padding-right + border-right + margin-right

Аналогично рассчитывается и высота элемента.

Свойство Display и типы элементов

Свойство display определяет тип отображения элемента. Основные значения:

1. Блочные элементы (display: block)

  • Занимают всю доступную ширину
  • Начинаются с новой строки
  • Можно задавать ширину/высоту, отступы
  • Примеры:

  • Нельзя задавать ширину/высоту, вертикальные отступы

3. Строчно-блочные элементы (display: inline-block)

  • Объединяют особенности блочных и строчных элементов
  • Занимают только необходимую ширину
  • Можно задавать ширину/высоту, отступы
  • Не начинаются с новой строки

Width и Height

Свойства width и height определяют размеры содержимого элемента. Важно помнить, что реальная ширина/высота блока складывается из:

  • Ширины/высоты содержимого
  • Внутренних отступов (padding)
  • Границ (border)
  • Внешних отступов (margin)

Пример:

.element {
    width: 200px;   /* Ширина содержимого */
    height: 100px;  /* Высота содержимого */
    padding: 20px;  /* Внутренние отступы */
    border: 2px solid black; /* Граница */
    margin: 10px;   /* Внешние отступы */
}

Общая ширина этого элемента: 10px (margin-left) + 2px (border-left) + 20px (padding-left) + 200px (width) + 20px (padding-right) + 2px (border-right) + 10px (margin-right) = 264px

Блок с width: 200px; height: 100px;

Padding

Padding - внутренние отступы между содержимым и границей элемента. Управляются свойствами:

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

Или сокращенным свойством padding:

/* 1 значение: все стороны */
padding: 10px;

/* 2 значения: вертикальные | горизонтальные */
padding: 10px 20px;

/* 3 значения: верх | горизонтальные | низ */
padding: 10px 20px 15px;

/* 4 значения: верх | право | низ | лево */
padding: 10px 20px 15px 5px;

Padding влияет на внутреннее пространство элемента и его общие размеры.

Блок с padding: 20px;

Border

Border - граница вокруг элемента, расположенная между padding и margin. Состоит из трех свойств:

border-width: 3px;      /* Толщина границы */
border-style: dashed;   /* Стиль границы */
border-color: red;      /* Цвет границы */

Или сокращенная запись:

border: 3px dashed red;

Основные стили границ:

  • solid - сплошная линия
  • dashed - пунктирная
  • dotted - точечная
  • double - двойная
  • groove - 3D желоб

Граница добавляется к общим размерам элемента.

Блок с border: 3px dashed red;

Margin - внешние отступы

Margin создает пространство вокруг элемента, отделяя его от соседних элементов. Основные особенности:

1. Положительные margin

Добавляют пространство между элементами:

Элемент 1
Элемент с margin: 20px;
Элемент 2

2. Отрицательные margin

Уменьшают пространство между элементами, могут даже перекрывать их:

Обычный блок
Блок с margin-top: -15px;
.negative-margin {
    margin-top: -15px; /* Сдвигает элемент вверх */
}

3. Схлопывание margin (margin collapsing)

Когда два вертикальных margin встречаются, они схлопываются в один больший margin:

Блок с margin-bottom: 30px;
Блок с margin-top: 20px;

Фактический отступ между блоками: 30px (больший margin), а не 50px.

Синтаксис

margin: 10px;            /* Все стороны */
margin: 10px 20px;       /* Вертикальные | горизонтальные */
margin: 10px 20px 15px;  /* Верх | горизонтальные | низ */
margin: 10px 20px 15px 5px; /* Верх | право | низ | лево */
