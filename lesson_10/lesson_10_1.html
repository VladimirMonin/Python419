Блочные элементы могут содержать другие блочные и строчные элементы, а строчные - только строчные.

Для изменения поведения элементов можно использовать CSS-свойства, такие как display, float и position.

Свойство display: inline-block

display: inline-block сочетает в себе характеристики блочных и строчных элементов. Элементы с этим свойством располагаются в одну строку, как строчные, но при этом позволяют задавать ширину, высоту, а также все внешние и внутренние отступы (margin и padding), как блочные элементы.

Это особенно полезно для создания горизонтальных меню, галерей изображений или других элементов, которые должны располагаться рядом, но при этом требовать контроля над их размерами и отступами.

Ограничения строчных элементов (display: inline)

Строчные элементы, такие как или , имеют следующие ограничения:

  • Нельзя задать width и height. Их размеры определяются содержимым.
  • Свойства margin-top и margin-bottom не влияют на вертикальное расположение элемента.
  • Свойства padding-top и padding-bottom увеличивают визуальный размер элемента, но не влияют на расположение соседних элементов по вертикали.

Внутренние и внешние отступы: Margin и Padding

В CSS, margin и padding являются ключевыми свойствами для управления пространством вокруг и внутри элементов, соответственно. Они являются частью блочной модели CSS, которая описывает, как элементы отображаются на веб-странице.

Padding (Внутренний отступ)

Свойство padding определяет пространство между содержимым элемента и его границей (border). Это внутренний отступ, который увеличивает размер элемента, но не влияет на расположение соседних элементов.

  • padding-top: Отступ сверху.
  • padding-right: Отступ справа.
  • padding-bottom: Отступ снизу.
  • padding-left: Отступ слева.

Сокращенная запись для Padding:

  • Одно значение: padding: 20px; — Применяет 20px ко всем четырем сторонам (сверху, справа, снизу, слева).
  • Два значения: padding: 10px 20px; — Применяет 10px к вертикальным сторонам (сверху и снизу) и 20px к горизонтальным сторонам (справа и слева).
  • Три значения: padding: 10px 20px 30px; — Применяет 10px сверху, 20px к горизонтальным сторонам (справа и слева), и 30px снизу.
  • Четыре значения: padding: 10px 20px 30px 40px; — Применяет 10px сверху, 20px справа, 30px снизу, и 40px слева (по часовой стрелке, начиная сверху).

Margin (Внешний отступ)

Свойство margin определяет пространство вокруг элемента, за его границей. Это внешний отступ, который используется для создания расстояния между элементами.

  • margin-top: Отступ сверху.
  • margin-right: Отступ справа.
  • margin-bottom: Отступ снизу.
  • margin-left: Отступ слева.

Сокращенная запись для Margin:

  • Одно значение: margin: 20px; — Применяет 20px ко всем четырем сторонам.
  • Два значения: margin: 10px 20px; — Применяет 10px к вертикальным сторонам (сверху и снизу) и 20px к горизонтальным сторонам (справа и слева).
  • Три значения: margin: 10px 20px 30px; — Применяет 10px сверху, 20px к горизонтальным сторонам (справа и слева), и 30px снизу.
  • Четыре значения: margin: 10px 20px 30px 40px; — Применяет 10px сверху, 20px справа, 30px снизу, и 40px слева (по часовой стрелке, начиная сверху).

Важное замечание: вертикальные отступы (margin-top и margin-bottom) у соседних блочных элементов могут схлопываться (margin collapsing), то есть больший отступ поглощает меньший, и итоговый отступ будет равен большему из двух.

Размеры по умолчанию в CSS

В CSS каждый HTML-элемент имеет определенное поведение по умолчанию в отношении своих размеров (ширины и высоты), которое зависит от его типа отображения (display).

  • Строчно-блочные элементы (display: inline-block): Эти элементы по умолчанию ведут себя как строчные в отношении ширины — они занимают только необходимую ширину для своего содержимого. Однако, в отличие от чисто строчных элементов, вы можете явно задать им width.

Высота (height) по умолчанию

  • Для всех типов элементов (блочных, строчных, строчно-блочных): Высота элемента по умолчанию определяется его содержимым. Если вы не зададите явную высоту, элемент будет расширяться по вертикали, чтобы вместить весь свой контент (текст, изображения, другие вложенные элементы).
  • Вы можете явно задать height для блочных и строчно-блочных элементов. Для строчных элементов свойство height не применяется.

Особые значения margin

  • margin: auto - автоматически центрирует элемент по горизонтали в родительском контейнере (если задана ширина). Это работает потому, что auto распределяет доступное пространство поровну слева и справа.

  • margin: 0 - полностью убирает внешние отступы у элемента. Это полезно для сброса стилей браузера по умолчанию. Часто используется в reset.css для устранения различий в отступах между браузерами.

Особенности margin: 0 и margin: auto

Свойство margin имеет два особых значения, которые часто используются в верстке:

  • margin: 0 - полностью убирает все внешние отступы у элемента. Это особенно полезно для сброса стилей по умолчанию у элементов типа body, h1-h6, p, ul и других.
  • margin: auto - автоматически распределяет доступное пространство вокруг элемента. Для горизонтального центрирования блочного элемента нужно задать ему ширину (width) и margin: auto.

Примеры:


/* Сброс отступов для всех элементов */
* {
  margin: 0;
  padding: 0;
}

/* Центрирование блочного элемента */
.container {
  width: 80%;
  margin: 0 auto; /* 0 - сверху и снизу, auto - слева и справа */
}
