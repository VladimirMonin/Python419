Знакомство с таблицами

Таблицы раньше использовались для позиционирования элементов на странице. Сейчас же их основное предназначение - это отображение табличных данных.

Перечень табличных тегов

  • <table>- этот тег отвечает за открытие таблицы
  • <tr>- этот тег отвечает за полосу в таблице
  • <td>- этот тег отвечает за ячейку в таблице
  • <tbody>- этот тег отвечает за тело таблицы (опционально)
  • <thead>- этот тег отвечает за заголовок таблицы (опционально)
  • <tfoot>- этот тег отвечает за подвал таблицы (опционально)
  • <th>- этот тег отвечает за ячейку заголовка таблицы
  • <caption>- этот тег отвечает за заголовок таблицы (опционально)

Пример простой таблицы 3 на 3

Чтобы таблица отображалась нормально, нам нужно использовать определенное количество CSS свойств.

  • border-collapse: collapse; - убирает двойные границы
  • border: 1px solid black; - добавляет визуально линии ячеек

Семантические теги в таблицах

Семантические теги в таблицах помогают лучше структурировать данные и делают их более доступными для поисковых систем и вспомогательных технологий.

  • <thead> - используется для определения заголовка таблицы.
  • <tbody> - используется для определения тела таблицы.
  • <tfoot> - используется для определения подвала таблицы.

Псевдо-классы и псевдо-элементы

Давайте разберемся с этими важными понятиями в CSS, которые позволяют нам стилизовать элементы в зависимости от их состояния или создавать виртуальные элементы.

Псевдоклассы (с одним двоеточием :)

Псевдоклассы используются для выбора элементов на основе их состояния или положения в структуре документа.

Псевдокласс Пояснение
:hover Применяет стили, когда пользователь наводит курсор мыши на элемент.
:active Применяет стили к элементу, который активируется пользователем (например, при нажатии кнопки мыши).
:focus Применяет стили к элементу, который получил фокус (например, при клике или навигации с клавиатуры).
:visited Применяет стили к ссылкам, которые пользователь уже посещал.
:link Применяет стили к непосещенным ссылкам.
:first-child Выбирает первый дочерний элемент своего родителя.
:last-child Выбирает последний дочерний элемент своего родителя.
:nth-child(n) Выбирает дочерний элемент по его порядковому номеру (n может быть числом, ключевым словом или формулой).
:not(selector) Выбирает элементы, которые не соответствуют указанному селектору.

Псевдоэлементы (с двойным двоеточием ::)

Псевдоэлементы позволяют нам стилизовать части элемента, которых на самом деле нет в HTML-коде, или создавать виртуальные элементы.

Псевдоэлемент Пояснение
::before Создает псевдоэлемент перед содержимым выбранного элемента. Часто используется с свойством content.
::after Создает псевдоэлемент после содержимого выбранного элемента. Часто используется с свойством content.
::first-line Применяет стили к первой строке текста в блочном элементе.
::first-letter Применяет стили к первой букве текста в блочном элементе.
::selection Применяет стили к части элемента, которая выделена пользователем (например, мышью).

Объединение столбцов и строк

Объединение столбцов и строк в таблицах позволяет создавать более сложные и структурированные таблицы.

Для объединения ячеек в таблице используются атрибуты colspan и rowspan.

Простое объединение строк

Объединение строк позволяет объединить несколько строк в одну. Это полезно, когда у вас есть данные, которые занимают несколько строк.
