Знакомство с таблицами

Таблицы раньше использовались для позиционирования элементов на странице. Сейчас же их основное предназначение - это отображение табличных данных.

Перечень табличных тегов

  • <table>- этот тег отвечает за открытие таблицы
  • <tr>- этот тег отвечает за полосу в таблице
  • <td>- этот тег отвечает за ячейку в таблице
  • <tbody>- этот тег отвечает за тело таблицы (опционально)
  • <thead>- этот тег отвечает за заголовок таблицы (опционально)
  • <tfoot>- этот тег отвечает за подвал таблицы (опционально)
  • <th>- этот тег отвечает за ячейку заголовка таблицы
  • <caption>- этот тег отвечает за заголовок таблицы (опционально)

Пример простой таблицы 3 на 3

Чтобы таблица отображалась нормально, нам нужно использовать определенное количество CSS свойств.

  • border-collapse: collapse; - убирает двойные границы
  • border: 1px solid black; - добавляет визуально линии ячеек

Семантические теги в таблицах

Семантические теги в таблицах помогают лучше структурировать данные и делают их более доступными для поисковых систем и вспомогательных технологий.

  • <thead> - используется для определения заголовка таблицы.
  • <tbody> - используется для определения тела таблицы.
  • <tfoot> - используется для определения подвала таблицы.

Псевдо-классы и псевдо-элементы

Давайте разберемся с этими важными понятиями в CSS, которые позволяют нам стилизовать элементы в зависимости от их состояния или создавать виртуальные элементы.

Псевдоклассы (с одним двоеточием :)

Псевдоклассы используются для выбора элементов на основе их состояния или положения в структуре документа.

Псевдокласс Пояснение
:hover Применяет стили, когда пользователь наводит курсор мыши на элемент.
:active Применяет стили к элементу, который активируется пользователем (например, при нажатии кнопки мыши).
:focus Применяет стили к элементу, который получил фокус (например, при клике или навигации с клавиатуры).
:visited Применяет стили к ссылкам, которые пользователь уже посещал.
:link Применяет стили к непосещенным ссылкам.
:first-child Выбирает первый дочерний элемент своего родителя.
:last-child Выбирает последний дочерний элемент своего родителя.
:nth-child(n) Выбирает дочерний элемент по его порядковому номеру (n может быть числом, ключевым словом или формулой).
:not(selector) Выбирает элементы, которые не соответствуют указанному селектору.

Псевдоэлементы (с двойным двоеточием ::)

Псевдоэлементы позволяют нам стилизовать части элемента, которых на самом деле нет в HTML-коде, или создавать виртуальные элементы.

Псевдоэлемент Пояснение
::before Создает псевдоэлемент перед содержимым выбранного элемента. Часто используется с свойством content.
::after Создает псевдоэлемент после содержимого выбранного элемента. Часто используется с свойством content.
::first-line Применяет стили к первой строке текста в блочном элементе.
::first-letter Применяет стили к первой букве текста в блочном элементе.
::selection Применяет стили к части элемента, которая выделена пользователем (например, мышью).

Объединение столбцов и строк

Объединение столбцов и строк в таблицах позволяет создавать более сложные и структурированные таблицы.

Для объединения ячеек в таблице используются атрибуты colspan и rowspan.

Объединение строк позволяет объединить несколько строк в одну. Это полезно, когда у вас есть данные, которые занимают несколько строк.

Большая сводная таблица всех CSS свойств которые мы прошли

Эта таблица содержит все CSS-свойства, которые мы изучили на уроках, а также некоторые популярные свойства, которые часто используются в современной веб-разработке.

Категория Название свойства Описание Возможные значения Изучено Пример использования Поддержка браузерами
Текст color Задает цвет текста. Это одно из базовых свойств CSS. Может принимать именованные цвета, HEX, RGB, HSL значения. Поддерживает прозрачность через RGBA и HSLA.
  • Именованные цвета (red, blue)
  • HEX (#FF0000)
  • RGB(255, 0, 0)
  • RGBA(255, 0, 0, 0.5)
 ✅ color: rgb(0, 0, 119); Все браузеры
text-align Определяет горизонтальное выравнивание текста в элементе. Очень часто используется для центрирования заголовков. Влияет на весь текстовый контент внутри элемента.
  • left (по умолчанию)
  • right
  • center
  • justify
 ✅ text-align: center; Все браузеры
text-decoration Добавляет или удаляет декоративные линии в тексте. Часто используется для удаления подчеркивания у ссылок. Может комбинироваться с другими свойствами декорации.
  • none
  • underline
  • line-through
  • overline
 ✅ text-decoration: underline; Все браузеры
text-shadow Добавляет тень к тексту. Позволяет создавать эффекты глубины и объема. Можно добавлять несколько теней через запятую.
  • horizontal-offset
  • vertical-offset
  • blur-radius
  • color
 ✅ text-shadow: 2px 2px 4px #000; Современные браузеры
Шрифты font-family Определяет семейство шрифта для текста. Можно указывать несколько шрифтов через запятую как запасные варианты. Последним обычно указывается общий тип шрифта.
  • Название шрифта
  • serif
  • sans-serif
  • monospace
 ✅ font-family: Arial, sans-serif; Все браузеры
font-size Задает размер шрифта. Может быть указан в различных единицах измерения. Наследуется дочерними элементами если не переопределен.
  • px (пиксели)
  • em (относительно родителя)
  • rem (относительно root)
  • % (проценты)
 ✅ font-size: 16px; Все браузеры
font-weight Определяет толщину шрифта. Можно использовать как числовые значения, так и ключевые слова. Не все шрифты поддерживают все значения.
  • normal (400)
  • bold (700)
  • 100-900
 ✅ font-weight: bold; Все браузеры
Блочная модель margin Задает внешние отступы элемента. Может быть задан как для всех сторон сразу, так и для каждой стороны отдельно. Поддерживает отрицательные значения.
  • px, em, rem
  • auto
  • %
 ✅ margin: 10px 20px; Все браузеры
padding Определяет внутренние отступы элемента. Увеличивает размер элемента. Не может принимать отрицательные значения. Часто используется для создания "воздуха" внутри элементов.
  • px, em, rem
  • %
 ✅ padding: 15px; Все браузеры
border Задает границу элемента. Является шорткатом для border-width, border-style и border-color. Может быть задан как для всех сторон, так и для каждой отдельно.
  • width style color
  • none
 ✅ border: 1px solid black; Все браузеры
Флексбокс display: flex Создает флекс-контейнер. Все его дочерние элементы становятся флекс-элементами. Позволяет создавать гибкие макеты. Является основой современной верстки.
  • flex
  • inline-flex
 ✅ display: flex; Современные браузеры
flex-direction Задает направление основной оси флекс-контейнера. Определяет, как будут располагаться флекс-элементы. От этого зависит работа justify-content и align-items.
  • row (по умолчанию)
  • column
  • row-reverse
  • column-reverse
 ✅ flex-direction: column; Современные браузеры
justify-content Определяет выравнивание элементов вдоль главной оси. Очень полезно для центрирования элементов. Работает в связке с flex-direction.
  • flex-start
  • center
  • space-between
  • space-around
 ✅ justify-content: center; Современные браузеры
Эффекты box-shadow Добавляет тень к элементу. Может создавать множественные тени. Часто используется для создания эффекта "поднятия" элемента. Поддерживает внутренние тени.
  • h-offset v-offset blur spread color
  • inset
 ✅ box-shadow: 0 0 10px rgba(0,0,0,0.5); Современные браузеры
border-radius Скругляет углы элемента. Можно задавать разные значения для каждого угла. Часто используется для создания круглых кнопок или аватарок.
  • px, em, rem
  • %
 ✅ border-radius: 10px; Современные браузеры
transition Создает плавный переход между состояниями элемента. Мощный инструмент для создания анимаций. Часто используется с :hover и другими псевдоклассами.
  • property
  • duration
  • timing-function
  • delay
 ✅ transition: all 0.3s ease; Современные браузеры
