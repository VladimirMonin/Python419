Знакомство с таблицами

Таблицы раньше использовались для позиционирования элементов на странице. Сейчас же их основное предназначение - это отображение табличных данных.

Перечень табличных тегов

  • <table>- этот тег отвечает за открытие таблицы
  • <tr>- этот тег отвечает за полосу в таблице
  • <td>- этот тег отвечает за ячейку в таблице
  • <tbody>- этот тег отвечает за тело таблицы (опционально)
  • <thead>- этот тег отвечает за заголовок таблицы (опционально)
  • <tfoot>- этот тег отвечает за подвал таблицы (опционально)
  • <th>- этот тег отвечает за ячейку заголовка таблицы
  • <caption>- этот тег отвечает за заголовок таблицы (опционально)

Пример простой таблицы 3 на 3

Чтобы таблица отображалась нормально, нам нужно использовать определенное количество CSS свойств.

  • border-collapse: collapse; - убирает двойные границы
  • border: 1px solid black; - добавляет визуально линии ячеек

Семантические теги в таблицах

Семантические теги в таблицах помогают лучше структурировать данные и делают их более доступными для поисковых систем и вспомогательных технологий.

  • <thead> - используется для определения заголовка таблицы.
  • <tbody> - используется для определения тела таблицы.
  • <tfoot> - используется для определения подвала таблицы.
